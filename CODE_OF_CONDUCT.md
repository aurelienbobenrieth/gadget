# Code of Conduct

## Our standard

We want this project to be welcoming, respectful, and useful to contributors of all experience levels.

Examples of expected behavior:

- being respectful in reviews and discussions
- assuming good intent while giving direct technical feedback
- focusing criticism on ideas, code, and decisions rather than people
- helping keep discussions constructive and on-topic

Examples of unacceptable behavior:

- harassment, personal attacks, or discriminatory language
- trolling, intimidation, or deliberate disruption
- publishing private information without permission
- any conduct that would make participation unsafe or hostile

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer privately through the repository owner profile.
