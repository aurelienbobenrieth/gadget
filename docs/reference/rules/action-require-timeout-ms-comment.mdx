Require a trailing inline comment describing the duration on the timeoutMS property in Gadget action options.

| Property | Value |
| --- | --- |
| Type | `suggestion` |
| Fixable | Yes (auto-fix available) |
| Recommended | `warn` |
| Strict | `error` |
| Scope | Gadget action files (global and model) |
| Category | Options and params |

## Reported messages

- **`missingComment`**: Add an inline duration comment to timeoutMS.
