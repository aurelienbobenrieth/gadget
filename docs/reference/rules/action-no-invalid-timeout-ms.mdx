Enforce that timeoutMS in Gadget action options does not exceed 900000ms (15 minutes).

| Property | Value |
| --- | --- |
| Type | `problem` |
| Fixable | Yes (auto-fix available) |
| Recommended | `error` |
| Strict | `error` |
| Scope | Gadget action files (global and model) |
| Category | Options and params |

## Reported messages

- **`exceedsMaximum`**: timeoutMS {{ value }} exceeds Gadget's max of 900000ms.
