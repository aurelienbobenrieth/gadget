Disallow maxConcurrency values exceeding Gadget's hard limit of 100 in api.enqueue queue options.

| Property | Value |
| --- | --- |
| Type | `problem` |
| Fixable | No |
| Recommended | `warn` |
| Strict | `error` |
| Scope | All files (any file containing `api.enqueue()` calls) |
| Category | Enqueue safety |

## Reported messages

- **`exceeded`**: maxConcurrency {{ value }} exceeds Gadget's limit of 100.

## Examples

### Incorrect

```ts
api.enqueue(api.publish, {}, { queue: { name: "q", maxConcurrency: 101 } })
```

### Correct

```ts
api.enqueue(api.publish, {}, { queue: { name: "q", maxConcurrency: 100 } })
```
