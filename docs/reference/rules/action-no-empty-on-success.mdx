Disallow empty onSuccess exports in Gadget actions.

| Property | Value |
| --- | --- |
| Type | `suggestion` |
| Fixable | Yes (auto-fix available) |
| Recommended | `warn` |
| Strict | `error` |
| Scope | Gadget action files (global and model) |
| Category | Runtime safety |

## Reported messages

- **`emptyOnSuccess`**: Remove empty onSuccess.

## Examples

### Incorrect

```ts
export const onSuccess: ActionOnSuccess = async () => {};
```

### Correct

```ts
export const onSuccess: ActionOnSuccess = async ({ record }) => {
  await doSomething(record);
};
```
