## Description

Disallow invalid actionType values in model action options.

## Reported messages

- `invalidActionType`: actionType "{{ value }}" must be create, update, delete, or custom.

## Invalid example

```ts
export const options: ActionOptions = { actionType: "scheduler" };
```

## Valid example

```ts
export const options: ActionOptions = { actionType: "create" };
```
