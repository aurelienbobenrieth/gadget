## Description

Enforce that the Gadget action options export contains only valid JSON-serializable literals.

## Reported messages

- `computedKey`: Option keys must be plain identifiers or string literals.
- `nonLiteralValue`: Option values must be JSON-serializable literals.
- `numericSeparator`: Numeric separator {{ raw }} is not valid JSON. Use {{ fixed }}.
- `shorthandProperty`: Shorthand properties are not allowed in options.

## Invalid example

```ts
export const options: ActionOptions = { timeoutMS: 90_000 };
```

## Auto-fix output

```ts
export const options: ActionOptions = { timeoutMS: 90000 };
```

## Valid example

```ts
export const options: ActionOptions = { returnType: true, timeoutMS: 50000 };
```
