Disallow returning a value from onSuccess in Gadget actions.

| Property | Value |
| --- | --- |
| Type | `suggestion` |
| Fixable | Yes (auto-fix available) |
| Recommended | `warn` |
| Strict | `error` |
| Scope | Gadget action files (global and model) |
| Category | Runtime safety |

## Reported messages

- **`noReturnValue`**: onSuccess return values are ignored. Use return; or remove it.

## Examples

<Tabs>
  <Tab title="Incorrect">

```ts
export const onSuccess: ActionOnSuccess = async ({ record }) => {
  return { success: true };
};
```

  </Tab>
  <Tab title="Auto-fix output">

  </Tab>
  <Tab title="Correct">

```ts
  await notify(record);
};
```
