Require a run export when onSuccess is exported in Gadget actions.

| Property | Value |
| --- | --- |
| Type | `problem` |
| Fixable | No |
| Recommended | `warn` |
| Strict | `error` |
| Scope | Gadget action files (global and model) |
| Category | Runtime safety |

## Reported messages

- **`onSuccessWithoutRun`**: onSuccess requires a run export.

## Examples

### Incorrect

```ts
export const onSuccess: ActionOnSuccess = async ({ record }) => {
  await doSomething(record);
};
```

### Correct

```ts
export const run: ActionRun = async ({ params }) => {};
export const onSuccess: ActionOnSuccess = async ({ record }) => {
  await doSomething(record);
};
```
