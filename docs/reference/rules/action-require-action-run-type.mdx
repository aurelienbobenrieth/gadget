Require Gadget action run and onSuccess exports to use generated type annotations.

| Property | Value |
| --- | --- |
| Type | `problem` |
| Fixable | Yes (auto-fix available) |
| Recommended | `warn` |
| Strict | `error` |
| Scope | Gadget action files (global and model) |
| Category | Runtime safety |

## Reported messages

- **`manualOnSuccessType`**: Use ActionOnSuccess for onSuccess.
- **`manualRunType`**: Use ActionRun for run.
- **`missingOnSuccessType`**: Type onSuccess as ActionOnSuccess.
- **`missingRunType`**: Type run as ActionRun.
