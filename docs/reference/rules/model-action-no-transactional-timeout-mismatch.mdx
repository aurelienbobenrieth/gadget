Disallow timeoutMS above 5000ms on model actions without explicitly setting transactional: false.

| Property | Value |
| --- | --- |
| Type | `problem` |
| Fixable | No |
| Recommended | `warn` |
| Strict | `error` |
| Scope | Model action files |
| Category | Options and params |

## Reported messages

- **`mismatch`**: timeoutMS {{ value }}ms has no effect on transactional model actions above 5000ms.

## Examples

### Incorrect

```ts
export const options: ActionOptions = { timeoutMS: 6000 };
```

### Correct

```ts
export const options: ActionOptions = { timeoutMS: 3000 };
```
